Auslößer

(sender)
  1. stellt ein oder mehere Events bereit
  2. ist eine konkrete Instanz einer Klasse
  3. sollte öffentlich sein

Events

  1. hat eine konkrete Quelle (sender)
  2. keinen dedizierten Empänger (subscriber)
  3. müßen vom Empänger aboniert werden
eher nicht wie sondern wie

Empfänger

(subscriber, reciver)
  1. stellt Funktionen die auf ein oder mehrere Events reagieren
  2. ist eine konkrete Instanz einer Klasse
